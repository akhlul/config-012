## PHP via Mise
this is to install php via mise for void-linux

```sh
sudo xbps-install -S base-devel pkg-config \
    autoconf bison curl gettext git openssl re2c \
    gd-devel libcurl-devel libedit-devel icu-devel \
    libjpeg-turbo-devel libmysqlclient-devel oniguruma-devel \
    libpng-devel postgresql-libs-devel readline-devel \
    sqlite-devel openssl-devel libxml2-devel libzip-devel \
    zlib-devel
```
